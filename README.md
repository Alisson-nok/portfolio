Portfólio - Alisson Sousa

Este é o meu primeiro projeto de portfólio desenvolvido para apresentar um pouco sobre mim, minhas habilidades e meus contatos profissionais.

O site foi criado utilizando HTML, CSS e JavaScript, com foco em praticar conceitos de desenvolvimento web e construir minha primeira página publicada na internet.

Acesse o site:
https://alisson-nok.github.io/portfolio/



Sobre o projeto

Este portfólio foi desenvolvido como parte do meu aprendizado em Desenvolvimento de Sistemas.
O objetivo foi aplicar na prática conceitos de:

Estruturação de páginas com HTML
Estilização e layout com CSS
Interatividade com JavaScript

O projeto também inclui animações e efeitos visuais para tornar a experiência do usuário mais agradável.



Funcionalidades

O site possui:

Seção Sobre mim
Cards interativos de habilidades
Animações e efeitos visuais
Fundo animado com partículas
Seção de contato
Links para GitHub e LinkedIn



Tecnologias utilizadas

HTML5
CSS3
JavaScript
Font Awesome
Particles.js
